# Mariangel777 Website

A multilingual website for blogger Mariangel777.

## ✨ Features
- Language support: English, Russian, Spanish, Japanese, Chinese, and Hindi.
- Clean and minimalist design focused on readability.
- Fully responsive layout for desktop and mobile devices.
- Quick access to social media.

## 🌍 Language Structure

The website supports multiple languages through separate directories:

- `/` — English (default)
- `/ru/` — Russian
- `/es/` — Spanish
- `/ja/` — Japanese
- `/zh/` — Chinese
- `/hi/` — Hindi

Each directory contains a translated version of the main page.

## 📁 Project Structure

- `css/` — Stylesheets
- `js/` — JavaScript files
- `assets/` — Images and icons
